## POTENTIAL SUBMISSION ISSUES & SOLUTIONS

### Issue 1: App Crashes on Review Device
**Solution**: 
- Fix crash, increment version code
- Rebuild and test
- Resubmit with new APK

### Issue 2: Ads Not Loading
**Solution**:
- Verify ad unit IDs correct
- Ensure app has Internet permission
- Test with Google test IDs first

### Issue 3: IAP Not Working
**Solution**:
- Verify product IDs match Google Play
- Ensure billing library integrated
- Test on device before submission

### Issue 4: Privacy Policy URL Invalid
**Solution**:
- Create privacy policy document
- Host on website
- Update URL in Play Console

### Issue 5: Screenshots Don't Meet Requirements
**Solution**:
- All screenshots must be 1080x1920 (portrait)
- No ads, watermarks, or external content
- Show actual app gameplay

### Issue 6: Permissions Rejected
**Solution**:
- Only request necessary permissions
- Clearly state why each permission needed
- Remove unused permissions from manifest
