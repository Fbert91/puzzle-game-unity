## ðŸŽ¨ VISUAL HIERARCHY

### Use Color to Guide Player Attention

```
Importance:  Color Usage:
HIGH         Bright primary colors (RED, YELLOW, BLUE)
MEDIUM       Secondary colors (GREEN, PURPLE, ORANGE)
LOW          Neutral colors (GRAY, WHITE)

Example game board:
- Active tiles:     Bright colors (get attention)
- Available tiles:  Medium colors (secondary focus)
- Locked tiles:     Gray (clearly unavailable)
- Background:       Neutral or very light (doesn't distract)
```

### Button Priority

```
PRIMARY BUTTON (e.g., "Play"):
- Color: YELLOW or BLUE
- Size: Largest
- Centered, prominent position

SECONDARY BUTTON (e.g., "Settings"):
- Color: Neutral or secondary color
- Size: Medium
- Less prominent position

TERTIARY BUTTON (e.g., "About"):
- Color: Gray or very subtle
- Size: Small
- Less prominent
```

## ðŸ“‹ ASSET COLOR ADJUSTMENTS

### If Kenney Assets Don't Match

Since Kenney assets have their own colors, you can:

1. **Keep Kenney colors** (easiest, consistent)
   - Kenney uses similar kid-friendly palette
   - Good color variety
   - Professional appearance

2. **Recolor in Unity**
   ```csharp
   image.color = new Color(0.31f, 0.80f, 0.77f); // Apply BLUE tint
   ```

3. **Adjust in image editor**
   - Use Photoshop/GIMP to recolor before import
   - Hue/Saturation adjustment
   - Color balance

4. **Create alternate sprites**
   - Use Sprites with multiple color variants
   - Swap between variants
   - Best for important assets like mascot

## ðŸŽ¯ ACCESSIBILITY NOTES

### Color Blind Friendly

- **Don't rely only on color** to convey meaning
- Add patterns or symbols to distinguish tiles
- Use high contrast between foreground/background
- Test with color-blindness simulators

### Contrast Ratios

- **Text on background**: 4.5:1 minimum for readability
- **UI elements**: 3:1 minimum
- Test with contrast checker tools

### Recommended Safe Combinations

```
âœ“ GOOD: Black text on Yellow background (high contrast)
âœ“ GOOD: White text on Blue background (high contrast)
âœ“ GOOD: Dark Gray text on Light Gray (medium contrast)

âœ— AVOID: Red text on Green (color-blind issues)
âœ— AVOID: Yellow text on White (low contrast)
âœ— AVOID: Blue text on Purple (similar hue)
```
