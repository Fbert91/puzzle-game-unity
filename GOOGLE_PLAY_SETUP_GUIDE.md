**Target**: Create Google Play developer account, set up app listing, fill all required information, upload APK and screenshots, and submit for review.  
**Time**: ~2-3 hours (includes waiting for review)  
**Cost**: $25 USD one-time developer account fee  
**Prerequisites**: Google account, APK built and tested, app icon + 2 screenshots  
**Result**: App submitted for review and approved within 1-3 hours

## STEP 1: CREATE GOOGLE PLAY DEVELOPER ACCOUNT

### Step 1.1: Go to Google Play Console

1. **Open**: https://play.google.com/console
2. **Sign in** with your Google account
3. Click **"Create account"** (if you don't have one)

### Step 1.2: Accept Terms & Pay Fee

1. **Accept** Google Play Developer Program Policies
2. **Enter Payment Information**:
   - Card details or PayPal
   - $25 USD one-time fee
3. Click **"Purchase"** to complete payment
4. **Wait** for processing (usually instant)

### Step 1.3: Complete Developer Profile

1. **Developer Name**: Enter your name or company name
   - Example: "Bert Games"
   - This shows on your app store listing
2. **Email Address**: Your contact email
3. **Phone Number**: (optional but recommended)
4. **Address**: Your location (required for tax info)
5. Click **"Complete"**

## STEP 2: CREATE APP LISTING

### Step 2.1: Start New App

1. **In Google Play Console**, click **"Create app"** button
2. **App Name**: Enter your game name
   - Example: "Puzzle Logic Pro"
   - Max 50 characters
   - This is what users see in Google Play store
3. **Default Language**: Select your language (e.g., English - United States)
4. **App Category**: Select **"Games"**
5. **Type**: Select **"Games"**
6. **Free or Paid**: Select **"Free"** (you'll monetize via ads/IAP)
7. Check **"I agree to the..."** checkboxes
8. Click **"Create app"**

### Step 2.2: Complete Required Information

After creating, you'll see your **App Dashboard**. You need to fill in:
- [ ] Product information (name, description, etc.)
- [ ] Pricing & distribution (countries)
- [ ] Ratings & content
- [ ] Screenshots & icon
- [ ] APK upload & release

## STEP 4: UPLOAD APP ICON & SCREENSHOTS

### Step 3.1: App Icon (Required)

**Icon Requirements**:
- Size: 512 x 512 pixels (PNG)
- Safe area: 192 x 192 pixels (center area visible)
- No transparency (fill with solid color)
- Format: PNG
- Max file size: 1 MB

**How to create icon** (if you don't have one):
1. Use free tools: Canva.com, Photopea.com, or GIMP
2. Design a simple, recognizable icon
3. Make it colorful and distinct
4. Center main element within 192x192 safe area
5. Export as 512x512 PNG (no transparency)

**Upload icon**:
1. **Go to**: Product Details ‚Üí App Information ‚Üí App Icon
2. **Click "Upload"** button
3. Select your 512x512 PNG file
4. Click **"Save"**

‚úÖ **App icon uploaded**

### Step 4.2: Feature Graphic (Recommended)

- Size: 1024 x 500 pixels (PNG or JPG)
- Used for app store banner
- Should showcase best features or main character
- Upload optional, but improves visibility

### Step 4.3: Screenshots (2-8 required, max 8)

**Screenshot Requirements**:
- Size: 1080 x 1920 pixels (phone vertical) OR 1920 x 1080 (horizontal)
- Format: PNG or JPG
- Minimum 2, maximum 8 screenshots
- Should show actual gameplay

**Recommended screenshots**:
1. **Gameplay screenshot**: Show main game board with tiles
2. **Level complete**: Show celebration/success screen
3. **Game menu**: Show main menu
4. (Optional) Mascot character close-up
5. (Optional) Difficulty levels or features

**How to capture screenshots**:

From your Android device:
1. **Build and install APK** on Android phone
2. **Launch the game**
3. **Take screenshot**:
   - Android: Press Power + Volume Down buttons
   - Screenshot saved to Photos
4. **Take 2-4 different screenshots** showing different game screens
5. **Transfer to computer** (USB or cloud)

**Upload screenshots**:
1. **Go to**: Product Details ‚Üí App Information ‚Üí Screenshots
2. Select **"Phone"** screenshots category
3. Click **"Upload"** ‚Üí Select image (1080x1920 recommended)
4. Add **text overlay** (optional):
   - Add captions like "Level 5: Beat the Challenge!"
   - Or leave as-is for clean look
5. Upload **2-8 screenshots**
6. Click **"Save"**

4. **Reviews policies**

## TROUBLESHOOTING

### Issue: "APK rejected - Cannot install on device"

**Cause**: APK build issue or incompatible Android target

**Solution**:
1. Rebuild APK (see ANDROID_BUILD_GUIDE.md)
2. Verify Android API level is 21+
3. Test APK on your own device first
4. Upload new version

### Issue: "Invalid app signature"

**Cause**: Keystore or signing issue

**Solution**:
1. Ensure same keystore used for all versions
2. Check keystore password is correct
3. Don't create new keystore for updates
4. Use same keystore for lifetime of app

### Issue: "App rejected for policy violation"

**Cause**: Violates Google Play policies

**Solution**:
1. Read rejection details carefully
2. Common issues:
   - Misleading description
   - Inappropriate content
   - Ads placed badly
   - Privacy policy missing
3. Fix issue
4. Re-submit

### Issue: "Screenshots not showing on store"

**Cause**: Screenshot size or format issue

**Solution**:
1. Verify screenshot size: 1080x1920 or 1920x1080
2. Use PNG or JPG format
3. Max 8 screenshots
4. Re-upload with correct size

### Issue: "App not appearing in search"

**Cause**: Too new (wait 24 hours) or regional restriction

**Solution**:
1. Wait 24 hours after approval (indexing)
2. Check if Canada is in selected countries
3. Try searching by exact app name
4. Wait more time, Google Play index updates hourly

### Issue: "Content rating not accepted"

**Cause**: Answers don't match app content

**Solution**:
1. Re-answer content rating questionnaire
2. Be honest about ad content
3. Resubmit questionnaire
4. Resubmit app for review
